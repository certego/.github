# Python linters

## How to use

W.I.P.

## Dependencies

- ### [Flake8](https://flake8.readthedocs.io/)
```bash
pip install flake8
```

- ### [Pylint](https://pylint.readthedocs.io/)
```bash
pip install pylint pylint-django
```

- ### [isort](https://isort.readthedocs.io/)
```bash
pip install isort
```

- ### [Black](https://black.readthedocs.io/)
```bash
pip install black
```
